## Manual Verification

Before submitting a Pull Request, please manually verify:

1. **Plugin Activation**: Ensure the plugin activates without errors in WordPress.
2. **Board Loading**: Check that the Scrum Board loads correctly in the admin dashboard.
3. **Task Operations**:
   - Create a new task.
   - Move it between columns.
   - Edit task details.
   - Assign a user.
4. **Console Errors**: Check the browser developer console for any JavaScript errors.
